# ATM-Python
ATM simulation with overdraft logic built in Python

## Features
* *Balance Management:* Tracks and updates account balance.
* *Overdraft Support:* Allows withdrawals exceeding the balance based on a set limit.
* *Transaction Loops:* Supports multiple operations in a single run.

## Usage
1. Run the script.
2. Enter the number of transactions.
3. Follow the on-screen prompts for withdrawal and overdraft options.
